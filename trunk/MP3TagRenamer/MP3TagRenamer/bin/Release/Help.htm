• Remove text from title
    Remove text from title that matchs this string or this regular expretion if checkbox for regullar expretion is checked.

    Ex 1:
    if you have titles like "U2 - 2002 - Zoorop - 02 - Numb" but you want only "Numb" using regulare expresions
    "U2 - 2002 - Zoorop - \d{2} - " or
    "U2 - \d{4} - Zoorop - \d{2} - "

    Ex. 2:
    You want to remove somethisn in the end of titles "U2 - 2002 - Zoorop - 02 - Numb -me" ("-me" is some kind of signature)
    " -me$" - reguler expretion takes only string from end of the line
